Как использовать

  1. Открыть IMPLEMENTATION_BLUEPRINT.md.
  2. Передать агенту DESIGN.md и tokens.json.
  3. Собрать оригинальный prototype: Home + Catalog + Product Detail + Cart.
  4. Не использовать бренд/логотипы/тексты Petrovich.
